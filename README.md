**Time taken to completion: In Progress**

## Unity features used:

* All Unity features from my previous 3 projects
* Gravity
* 2D Box/Circle/Polygon colliders & collision
* Bounds
* Rigid Body 2D
* Particle effects
* Sound effects
* Background Music
